# LUMINA - Plataforma de Aprendizaje de Programación

LUMINA es una aplicación avanzada y gamificada diseñada para desarrolladores de élite. Inspirada en la estética de herramientas modernas como Raycast y Trae.ai, ofrece una experiencia de aprendizaje inmersiva para dominar lenguajes como Python, SQL y Desarrollo Web.

## 🚀 Características

- **Diseño Premium**: Interfaz oscura, minimalista y fluida con efectos de desenfoque y gradientes.
- **Aprendizaje Interactivo**: Lecciones dinámicas con un editor de código integrado (Monaco Editor).
- **Gamificación**: Sistema de niveles, racha de días, XP y gemas para desbloquear recompensas.
- **Maestría**: Un catálogo de cursos con navegación horizontal tipo "sticky scroll" (inspirado en trae.ai).
- **Identidad Digital**: Personalizador de avatares y perfiles detallados.
- **Tienda de Recompensas**: Intercambia gemas por potenciadores y mejoras estéticas.

## 🛠️ Tecnologías

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS
- **Animaciones**: Framer Motion
- **Estado**: Zustand
- **Scroll**: Lenis (Smooth Scroll)
- **Editor**: Monaco Editor

## 📦 Instalación

1. Clona el repositorio.
2. Instala las dependencias:
   ```bash
   npm install --legacy-peer-deps
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
